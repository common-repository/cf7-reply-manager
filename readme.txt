CF7 add-on to provide easy, pre-compiled answers for each form in your website

== Description ==

Forget about wasting your time by manually answering every CF7 submission!
CF7 Replying Manager it’s the quick and easy Contact Form 7 add-on to reply to every form submission of your website: just set a couple of answers (approval/disapproval), decide an attachment for each, and that’s it 

Furthermore, you can benefit from others functions like filtering by state (pending/approved/rejected) and a searching box tool always available.
Download CF7 Replying Manager, it’s free!!

== Installation ==
1. Ensure you have istalled Contact Form 7.
2. Upload the plugin files to the `/wp-content/plugins/cf7-reply-manager` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the Cf7 Reply Manager->Settings screen to configure the plugin.
